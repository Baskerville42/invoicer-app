# Security Policy

## Supported Versions

Ми активно підтримуємо останню стабільну версію проекту.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Ми серйозно ставимося до безпеки нашого додатку. Якщо ви виявили вразливість, будь ласка, дотримуйтесь наступної процедури:

1. **НЕ створюйте публічне Issue** для вразливостей безпеки.
2. Надішліть детальний опис вразливості на електронну пошту адміністратора (вкажіть контактну адресу, якщо вона є, або повідомте через приватні канали).
3. Ми підтвердимо отримання вашого звіту протягом 48 годин.
4. Ми повідомимо вас про прогрес у виправленні вразливості.

Дякуємо за допомогу в забезпеченні безпеки нашого проекту!
